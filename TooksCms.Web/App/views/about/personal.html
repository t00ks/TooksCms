TooksCMS

The only personal project that I currently have underway is the site you are currently looking at. It is a SQL driven content management system (CMS) built on the Microsoft stack sporting a single page DurandalJS based front end.

I started writing this CMS as a personal project to advance my knowledge. So bearing that in mind the code is a bit of a liquorish all sort of technologies and coding patterns! I hope that the newer code is of a better standard than the older code, but never the less some of the older stuff I’m still pretty happy with. That said, some of my older ideas are currently haunting me a little when trying to improve the overall quality of the project.

Featues

  • Customisable Articles
  • Review Article Types with Customisable Criteria
  • Image Galleries
  • Tags
  • Comments
  • Code Snippets
  • YouTube and Vimeo Support
  • Area Configurable Sidebar with Twitter Integration
  • Site Access Statistics
  • Site Search
  • Authenticated Administration Section with User Accounts
  • Customisable Static Routing
