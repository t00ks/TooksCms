About

Welcome to my website, this is my technical playground where I will post about anything that takes my fancy. This website has been written by me, from scratch. Why, I hear you ask, when there are loads of blog engines available do I write my own? Because I can.

I am a software developer with a few years in the industry and have been very slowly writing this site as a side line to my day job to further develop my skills in all that is new and flashy in the glamorous world of web development.

So now that I have created this site, I’m going to use it to bug the world with my views on things like video games, films and music. I will post code snippets and attempt to be helpful to those who are struggling with issues I have already struggled with and overcome.

I used to run a site called PC Rules back in my youth which has over the past 5 or so years died a bit of a death. However still even now if you search for Saitek R440 in Google we are still in the top 3 (previously number 1) in the search results. My long term plan is to port all the old reviews into the new engine and point pcrules.com to this site.

JavaScript Libraries and Plugins

jQuery
jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript.
jQuery UI
jQuery UI provides abstractions for low-level interaction and animation, advanced effects and high-level, themeable widgets, built on top of the jQuery JavaScript Library, that you can use to build highly interactive web applications.
Knockout JS
Knockout is a JavaScript library that helps you to create rich, responsive display and editor user interfaces with a clean underlying data model. Any time you have sections of UI that update dynamically (e.g., changing depending on the user’s actions or when an external data source changes), KO can help you implement it more simply and maintainably.
SyntaxHighlighter
SyntaxHighlighter is a fully functional self-contained code syntax highlighter developed in JavaScript
